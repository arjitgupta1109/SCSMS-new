# 🏙️ SCSMS – Smart Civic Service Management System

A full-stack web application developed to simplify the process of reporting, tracking, and resolving civic issues. The platform connects citizens, administrators, and municipal workers through a centralized complaint management system.

---

## Features

- Citizen registration and login
- Role-based authentication (Citizen, Admin, Worker)
- Online complaint submission with image upload
- AI-assisted complaint title suggestion
- Complaint assignment to workers
- Real-time complaint status tracking
- Live chat support using Socket.IO
- Email OTP verification
- Notifications for complaint updates
- Worker management dashboard
- Admin announcements
- Complaint analytics and activity tracking

## Installation

### Clone the repository

```bash
git clone https://github.com/arjitgupta1109/SCSMS-new.git
```

### Install backend dependencies

```bash
cd backend
npm install
```

### Start the backend

```bash
npm start
```
